PlanetLoopTimer 専用

このアプリについて

PlanetLoopTimer は子供向けタイマーアプリです。第三者広告を一切表示せず、外部との通信も行いません。お子さんのデータは端末内に保存され、外部に送信されることはありません。

収集する情報

アプリの設定や最長記録などは、お使いの端末内にのみ保存されます。当方のサーバーや外部サービスにこれらの情報を送信することはありません。

個人を特定する情報 (氏名・メールアドレス・電話番号・端末識別子など) は一切取得しません。

広告について

本アプリには広告を一切表示しません。第三者広告も、自社の他アプリの宣伝広告も、お子さんの画面・保護者画面のいずれにも表示されません。

解析ツール / トラッキング

利用状況を集計する解析ツールは使用しません。広告識別子 (IDFA) やトラッキング ID の取得・送信も行いません。

第三者への提供

ユーザーの情報を第三者に販売・提供することはありません。そもそも、外部に送信できる情報を取得していません。

お問い合わせ

ご質問・ご要望があれば、App Store のアプリページから「デベロッパ Web サイト」経由でご連絡ください。

改定

本ポリシーは必要に応じて改定することがあります。重要な変更がある場合は、このページ上でお知らせします。
